# 🏋️‍♂️ TrainerApp

A modern web application built for **personal trainers and clients** to organize, track, and optimize training programs.  
TrainerApp provides an intuitive interface for building workout plans, logging progress, and visualizing performance data.

## 🚀 Features
- 🧍‍♂️ Create and manage workout programs  
- 💪 Track exercises, sets, and reps  
- 📅 Plan training days and rest days  
- 📈 View client or personal progress over time  
- ⚙️ (Coming soon) Save user data to backend database  
- 🔐 (Coming soon) Authentication for trainers and clients

## ⚙️ Installation

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/OdinsBeard82/TrainerApp.git

# 2. Navigate into the folder
cd TrainerApp

# 3. Install dependencies
npm install

# 4. Start the development server
npm start

The app will be available at http://localhost:3000
